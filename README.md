# Fairplay Rail

Fairplay Rail is a web-based railway ticket booking prototype that demonstrates a structured multi-step booking flow with a modern UI, session-based data transfer, a behavioral status indicator, and an admin dashboard.

## Features

- Train listing and selection page.
- Passenger details form.
- Payment and confirmation flow.
- Session storage for preserving booking data across pages.
- Behavioral activity status indicator.
- Admin dashboard to view the latest booking details.
- Responsive dark-themed UI with glassmorphism styling.

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Browser Session Storage

## Project Flow

1. Home page with project introduction.
2. Train selection page to browse available trains.
3. Passenger details page to enter traveler information.
4. Payment page to review fare and proceed.
5. Confirmation page showing final ticket details.
6. Admin dashboard for monitoring the latest booking.

## Behavioral Status Indicator

The project includes a lightweight interaction-based status display that reacts to user activity such as:
- Mouse movement
- Clicks
- Scrolling
- Typing
- Window focus changes

This feature is included as a demonstration of interface-level activity monitoring.

## How to Run

1. Download or clone the project.
2. Open `index.html` in a browser.
3. Navigate through the booking flow step by step.

## Notes

- This project is a frontend prototype.
- Booking data is stored temporarily in browser session storage.
- The admin dashboard displays the latest booking record available in the current session.
- The behavioral status indicator is a demo feature and not a full security system.
